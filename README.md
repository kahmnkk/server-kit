# Nodejs SERVER-KIT

nodejs server starter kit.

## Includes

-   express
-   socket-io
-   mysql
-   redis
-   log service

## VS Code Prettier Setting

-   Search 'Prettier - Code formatter' in VS Code Extension
-   Go to VS Code Setting(ctrl + ,)
-   Set Default Formatter to 'prettier-vscode'
-   Check 'Format On Save'
